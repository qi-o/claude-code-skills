# Spatial Design Reference

## Spacing Scale

```css
--space-1: 0.25rem;   /* 4px */
--space-2: 0.5rem;    /* 8px */
--space-3: 0.75rem;   /* 12px */
--space-4: 1rem;      /* 16px */
--space-5: 1.5rem;    /* 24px */
--space-6: 2rem;      /* 32px */
--space-8: 3rem;      /* 48px */
--space-10: 4rem;     /* 64px */
--space-12: 6rem;     /* 96px */
--space-16: 8rem;     /* 128px */
```

## Layout Principles

### Whitespace Usage
- Generous whitespace = luxury/premium feel
- Tight spacing = density/information-heavy
- Consistency is key

### Grid Systems
- 12-column grid most common
- Gutters: 16px, 24px, 32px
- Margins: 16px (mobile), 24px (tablet), 48px+ (desktop)

### Responsive Breakpoints
```css
--breakpoint-sm: 640px;
--breakpoint-md: 768px;
--breakpoint-lg: 1024px;
--breakpoint-xl: 1280px;
--breakpoint-2xl: 1536px;
```

## Visual Hierarchy
- Size: larger = more important
- Position: top-left = most visible
- Color: contrast draws attention
- Spacing: isolated = important

## Alignment
- Left align text (most readable)
- Use grids for alignment
- Align related elements
- Create visual rhythm
