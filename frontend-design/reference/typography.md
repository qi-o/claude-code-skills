# Typography Reference

## Font Selection Principles

### Display Fonts (Headlines)
- Choose distinctive, characterful fonts
- Avoid: Inter, Roboto, Arial, system defaults
- Consider: Playfair Display, Montserrat, Poppins, Oswald, Raleway, Cormorant, etc.

### Body Fonts
- Prioritize readability
- Ensure good x-height
- Test at multiple sizes

### Font Pairing
- Contrast weights: bold display + regular body
- Contrast styles: serif display + sans-serif body
- Match moods: both fonts should complement each other

## Typography Scale

```
--font-xs: 0.75rem;    /* 12px */
--font-sm: 0.875rem;   /* 14px */
--font-base: 1rem;     /* 16px */
--font-lg: 1.125rem;   /* 18px */
--font-xl: 1.25rem;    /* 20px */
--font-2xl: 1.5rem;    /* 24px */
--font-3xl: 1.875rem;  /* 30px */
--font-4xl: 2.25rem;   /* 36px */
--font-5xl: 3rem;      /* 48px */
--font-6xl: 3.75rem;   /* 60px */
```

## Line Height Guidelines
- Headlines: 1.1 - 1.2
- Body text: 1.5 - 1.6
- Dense content: 1.4
- Comfortable reading: 1.6 - 1.8

## Letter Spacing
- Headlines: -0.02em to 0
- Body: 0
- Caps: 0.05em to 0.1em

## Best Practices
- Use relative units (rem, em)
- Set base font size on root
- Limit character count per line (45-75 chars ideal)
- Ensure sufficient contrast
